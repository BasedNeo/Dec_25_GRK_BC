CORE DESIGN RULES:
═══════════════════════════════════════════════════════════════════

1. MAXIMIZE SCREEN USAGE
   - Game fills 100% of available space
   - UI overlays the game (transparent backgrounds)
   - NO wasted space, NO unnecessary margins
   - Desktop: Full browser viewport
   - Mobile: Edge-to-edge, full screen

2. VIBRANT COLOR SCHEME
   Background: Deep black (#050510)
   Primary: Electric Cyan (#00FFFF) 
   Secondary: Hot Magenta (#FF00FF)
   Accent: Neon Green (#00FF88)
   Energy: Solar Orange (#FF6B00)
   All colors GLOW: box-shadow: 0 0 20px [color]

3. CRISP GRAPHICS
   - Sharp vector shapes (no blurry images)
   - 60fps animations (use CSS transform only)
   - High contrast (bright on dark)
   - Clean geometric designs
   - Particle effects for energy/impact

4. TOUGH CHALLENGES
   - Difficulty increases every level
   - Fast reflexes required
   - One mistake matters
   - "Just one more try" addictive
   - Leaderboard-worthy scores
