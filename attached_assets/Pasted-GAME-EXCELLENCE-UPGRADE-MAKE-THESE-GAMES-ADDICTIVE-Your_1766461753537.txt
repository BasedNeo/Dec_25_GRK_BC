   - Tiny shake (2-3px) on impacts, explosions, collisions

   // Simple screen shake
   const shake = (intensity = 5, duration = 100) => {
     element.style.transform = `translate(${Math.random()*intensity}px, ${Math.random()*intensity}px)`;
     setTimeout(() => element.style.transform = '', duration);
   };
   2. PARTICLE EFFECTS

   - Use navigator.vibrate() for mobile devices
