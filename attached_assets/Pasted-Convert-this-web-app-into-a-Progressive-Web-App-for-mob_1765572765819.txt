SERVICE WORKER (sw.js):
- Cache static assets (HTML, CSS, JS, images)
- Cache-first strategy for assets
- Network-first strategy for API calls
- Offline fallback page
- Background sync for failed requests

IMPLEMENTATION:
1. Create manifest.json with above content
2. Add to HTML: <link rel="manifest" href="/manifest.json">
3. Add meta tags:
   <meta name="theme-color" content="#6366f1">
   <meta name="apple-mobile-web-app-capable" content="yes">
   <meta name="apple-mobile-web-app-status-bar-style" content="black-translucent">
   <link rel="apple-touch-icon" href="/icons/icon-192.png">

4. Register service worker in main JS:
   if ('serviceWorker' in navigator) {
     navigator.serviceWorker.register('/sw.js');
   }

5. Add install prompt handling:
   - Listen for beforeinstallprompt event
   - Show custom "Add to Home Screen" banner
   - Track installation analytics

PUSH NOTIFICATIONS (optional):
- Request notification permission

- Send notification for legendary mint alerts

OFFLINE SUPPORT:
- Cache last known stats
- Show "Offline - showing cached data" banner
- Queue any user actions for when back online
